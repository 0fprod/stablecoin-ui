## Decentralized Stablecoin UI

This is a simple UI for the Decentralized Stablecoin project. It allows users to deposit, redeem collaterals and mint stablecoins. The UI is built using React and interacts with the smart contracts deployed on the Sepolia testnet.

## How to run the project

1. Clone the repository
2. Run `npm install` to install the dependencies
3. Run `npm dev` to start the development server
4. Open `http://localhost:5173` in your browser

## How to use the UI

1. Connect your wallet
2. Deposit collaterals
3. Mint stablecoins
4. Liquidate positions
